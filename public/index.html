Vocdoni Proof of Concept on (Purish) JavaScript

This project is a Demo on how to use the Vocdoni SDK in a simple way, for integrating with Decidim.

For making this work, you'll need the MetaMask browser addon.

  1. Download and install the MetaMask addon for your browser or Operating System.
  2. Set up a wallet
  3. Once its set up, change the Network to test network. On the Firefox addon, this is done by:
    1. clicking "Ethereum Mainnet" Networks in the top right
    2. clicking on "Show/hide test networks"
    3. clicking on "Show test networks" -> "ON"

Open your web development tools to see debug messages.
